### 2. git window & mac 협업
**LF, CRLF 관련 오류**
서로 다른 OS간 git 사용때문에 발생하는 오류다 정확한것은 
OS에 맞는 설정을 구글에 꼭 찾아보길!
.gitconfig 설정
